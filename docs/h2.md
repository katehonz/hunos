# HTTP/2 Support (`hunos/h2`)

Experimental HTTP/2 over cleartext (h2c) support for Hunos.

## Features

- HTTP/2 frame parser (HEADERS, DATA, SETTINGS, PING, GOAWAY)
- HPACK header compression/decompression
- Stream multiplexing
- Server push (experimental)
- H2C Upgrade negotiation

## Usage

HTTP/2 is negotiated automatically via the `Upgrade: h2c` header or can be forced.

```nim
import hunos, hunos/h2

# The core server includes HTTP/2 negotiation.
# No additional setup is required for basic h2c support.

let server = newServer(handler)
server.serve(Port(8080))
```

## Direct H2C

For direct HTTP/2 connections without upgrade:

```nim
import hunos/h2

# Use H2Connection for low-level frame handling
var conn = newH2Connection(socket)
conn.sendSettings()
# ... handle frames
```

## Limitations

- TLS + ALPN is not yet implemented (h2 only, no h2s)
- Stream prioritization is not implemented
- Flow control is basic

## Testing

Run the HTTP/2 tests:

```bash
nim c --threads:on --mm:orc --path:src -r tests/test_h2.nim
nim c --threads:on --mm:orc --path:src -r tests/test_h2_integration.nim
```
